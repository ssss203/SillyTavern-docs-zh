# 👥 多用户模式（Multi-user mode）
> 💡 **肆说**：多用户模式——一台服务器多人用。个人使用不需要看。

允许多人共用一个 SillyTavern 服务器，每个用户有独立的设置、扩展和数据。用户账号可设置密码保护。

在 `config.yaml` 中设置 `enableUserAccounts: true` 启用。角色分 Admin（可管理用户和全局扩展）和 User（仅管理自己）。支持密码恢复、内容脚手架等功能。

📖 原文链接：[Multi-user mode](https://docs.sillytavern.app/administration/multi-user/)
