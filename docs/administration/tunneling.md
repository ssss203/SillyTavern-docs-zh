# 🌐 VPN 和隧道（VPNs and Tunneling）
> 💡 **肆说**：想让外面的人访问你的SillyTavern？用隧道服务，比端口转发安全多了。

安全地从外网访问本地 SillyTavern 实例的方法：

1. **家庭 VPN** - 路由器自带 OpenVPN/WireGuard
2. **Cloudflare Zero Trust** - 免费，支持50用户
3. **独立 Cloudflare/ngrok 隧道** - 简单但每次换链接
4. **Tailscale** - 免费 VPN，设置简单

📖 原文链接：[VPNs and Tunneling](https://docs.sillytavern.app/administration/tunneling/)
