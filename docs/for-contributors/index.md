# 🛠️ 开发与自动化（Development and Automation）
> 💡 **肆说**：你不是开发者，跳过。这些是给想给SillyTavern贡献代码的人看的。

SillyTavern 的默认包只提供基础平台，可通过多种方式扩展：

- **STscript** - 基于批处理聊天命令的脚本语言，无需编程知识
- **Function Calling** - 让 LLM 使用外部数据源或触发扩展功能
- **UI Extensions** - 浏览器环境运行的扩展，通过事件和 API 扩展功能
- **Server Plugins** - NodeJS 环境运行的插件，可添加新 API 端点
- **Internationalization (i18n)** - 翻译 SillyTavern UI 的方法

📖 原文链接：[Development and Automation](https://docs.sillytavern.app/for-contributors/)
