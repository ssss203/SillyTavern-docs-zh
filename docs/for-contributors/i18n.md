# 🌐 国际化（i18n / Internationalization）
> 💡 **肆说**：国际化开发文档，做角色卡暂时跳过。

SillyTavern 支持多语言。翻译文件为 JSON 格式，存放在 `public/locales/` 中。使用 `data-i18n` HTML 属性和 `t` 模板字符串函数实现翻译。调试工具在 User Settings > Debug Menu 中。添加新语言需创建 `public/locales/lang.json` 条目和对应语言 JSON 文件。

📖 原文链接：[Internationalization (i18n)](https://docs.sillytavern.app/for-contributors/i18n/)
