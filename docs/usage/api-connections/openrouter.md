# 🔀 OpenRouter
> 💡 **肆说**：OpenRouter是"AI模型的万能接口"——一个账号用几十种模型，不用每个都注册。国内需要梯子，但很方便。

OpenRouter 同时作为 Text Completion 和 Chat Completion 来源可用。所有模型都可通过任一 API 访问，但功能可能不同（如图像内联和工具调用仅在 Chat Completion API 可用）。

不想注册十几个 API 服务，又想访问所有最新模型？用 OpenRouter。

OpenRouter 让你用一个端点访问 DeepSeek、Claude、Gemini 等模型，共用一个信用池。

有免费试用（约 $1），之后付费。无需订阅，用多少付多少。部分模型有每日有限免费请求。
> 💡 **肆说**：部分模型免费，可以先试试。充$10就能永久获得免费模型的每日额度。

要永久获得慷慨每日额度的免费模型，需至少一次性购买 $10 信用。

详见 [OpenRouter FAQ](https://openrouter.ai/docs/faq)。

**设置步骤：**
1. 选择 Chat Completion API
2. 选 OpenRouter 作为来源
3. 点击 "Authorize" 通过 OAuth 获取密钥，或从 [这里](https://openrouter.ai/keys) 生成 API 密钥粘贴
4. 点击 "Connect" 并选择模型
5. 可选：用 "Test Message" 验证连接
