# 👥 AI Horde
> 💡 **肆说**：免费！不用梯子！但质量随机（取决于志愿者的GPU），排队时间长，适合零成本体验。

## 免责声明

- AI Horde 是众包分布式 GPU 集群，完全由志愿者运行
- 默认匿名发送输入，运行 Horde Worker 的人看不到回复
- 但恶意 Worker 可能修改代码来记录你的活动或产生不良回复
- **永远不要**通过 Horde 发送个人信息
> 💡 **肆说**：再次提醒！Horde是志愿者跑的，理论上有人能看到你发的内容，别发隐私信息。

开启"仅信任 Worker"复选框可限制为只使用经过信任的长期 Worker。

SillyTavern 有内置功能：当 Horde Worker 生成回复时，ST 会记录 Worker ID 和使用的模型。鼠标悬停在聊天条目上可查看。

## 设置

- SillyTavern 开箱即可连接 Horde，无需额外设置
- 在 ST API 面板选择 "AI Horde"
- 从底部的模型选择器选一个或多个模型
- 选择角色开始聊天

默认连接到 Horde 的低优先级"访客账号"，等待时间可能很长。
> 💡 **肆说**：注册个免费账号能把优先级提高，等的时间短很多。

## 减少等待时间的技巧

- 在 [Horde 网站](https://aihorde.net/register) 注册账号，把密钥填入 ST
- 设置一个 [Horde Worker](https://github.com/Haidra-Org/AI-Horde-Worker#readme) 贡献你的 GPU。贡献 GPU 赚取 'Kudos'（Horde 专用货币），Kudos 越多回复越快
