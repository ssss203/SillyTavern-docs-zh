# 🌙 DreamGen
> 💡 **肆说**：专门做角色扮演和故事写作的API，有免费额度。国内需要梯子。

DreamGen 是 AI 角色扮演和故事写作的 API。有免费额度和付费订阅。

## 连接 DreamGen

### 获取 API 密钥
1. 去 [DreamGen API Keys](https://dreamgen.com/account/api-keys) 页面
2. 点击 "New API Key"
3. 复制密钥

### 连接
1. SillyTavern 连接设置选 API: Text Completion
2. 选 API Type: DreamGen
3. 输入 API 密钥
4. 可选：选择模型

## 模型

- **Lucid Max**（API 中叫 `lucid-v1-max`）- 更有创意，处理复杂指令更好
- **Lucid Base**（API 中叫 `lucid-v1-base`）- 消耗更少信用，更快

## 设置

强烈推荐使用官方预设：
- [DreamGen Lucid V1 Role-Play preset](https://dreamgen.com/docs/models/lucid-v1/sillytavern/master-preset/role-play)
- [DreamGen Lucid V1 Story preset](https://dreamgen.com/docs/models/lucid-v1/sillytavern/master-preset/story)

确保启用 Instruct Mode 并导入时选中所有复选框。
