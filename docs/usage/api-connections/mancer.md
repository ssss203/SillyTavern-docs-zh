# 🎭 Mancer
> 💡 **肆说**：不审查内容的API服务，有一个免费模型。国内需要梯子。

Mancer 是一个大语言模型推理服务，让你跑任何提示词且不审查回复。大多数模型需要预充值，但有一个免费模型。

- [模型列表](https://mancer.tech/models)
- [价格](https://mancer.tech/pricing)

## 开始使用

1. 在 [mancer.tech](https://mancer.tech/) 注册账号
2. 点击 Dashboard 复制 API Key
3. 在 SillyTavern 选 Text Completion API，选 Mancer
4. 输入 API Key，点击 Connect

## 匿名日志

如果你不介意聊天可能被用于训练模型，可以开启匿名日志获得 25% token 折扣。
