# 📖 使用指南
> 💡 **肆说**：这是SillyTavern的完整使用指南目录。建议先看"快速开始"，然后按需看其他页面。

用你的方式跟 AI 互动。构建你的世界、你的作品、你的梦想。

## 🚀 入门

- **[快速开始](quick-start.md)** - 发送第一条消息并获得 AI 回复
- **[聊天操作](chatting.md)** - 如何跟 AI 聊天及使用聊天界面
- **[常见问题 FAQ](faq.md)** - 关于 SillyTavern、AI 模型、创建角色、获得更好回复等的常见问题

## 📚 基础知识

- **[API 连接](api-connections/index.md)** - 连接 AI 模型来生成文本、图片等
- **[角色卡](characters.md)** 和 **[人设](core-concepts/personas.md)** - 创建和使用角色来塑造 AI 的角色，用人设来定义你的身份
- **[回复配置](common-settings.md)** 和 **[提示词](prompts/index.md)** - 控制你发给 AI 的请求以及它如何回复
- **[欢迎页助手](welcome-assistants.md)** - 设置欢迎页助手，在欢迎界面用指定角色迎接你

## 🏗️ 在 SillyTavern 上构建

- **[世界信息 World Info](core-concepts/worldinfo.md)** - 管理信息以及何时插入到提示词中
- **[数据银行 Data Bank](core-concepts/data-bank.md)** - 存储和检索信息供 AI 回复使用
- **[扩展插件](../extensions/index.md)** - 给 AI 或界面添加新功能
- **[开发与自动化](../for-contributors/index.md)** - 自动化任务、让 AI 与世界互动、编写自己的扩展

## 🎛️ 控制面板

从左到右，每个按钮的作用：

- **[回复配置](common-settings.md)** 和 **[提示词管理器](prompts/prompt-manager.md)** - 控制文本生成和采样。自定义 Chat Completion API 的提示词构建
- **[API 连接](api-connections/index.md)** - 连接 AI 模型
- **[高级格式化](core-concepts/advancedformatting.md)** - 自定义 Text Completion API 的提示词构建
- **[世界信息](core-concepts/worldinfo.md)** - 管理信息及插入时机
- **[用户设置](user-settings/index.md)** - 更改主题、消息和聊天的外观
- **扩展插件** - 添加新功能
- **[人设](core-concepts/personas.md)** - 创建和管理人设
- **[角色](characters.md)** - 创建和管理角色
