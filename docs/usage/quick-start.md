# 🚀 快速开始
> 💡 **肆说**：零基础？从这里开始！跟着做5分钟就能跟AI聊天。

> "我什么都不懂，直接给我最简单最快的上手方式。" —— *匿名用户*

只需几分钟就能开始使用 SillyTavern。两种简单方式：

- 你可以用 **AI Horde** 免费。AI Horde 是一个社区驱动的 AI 服务，提供多种 AI 模型。
- 如果你有 **OpenAI** 账号或想注册一个，可以用 OpenAI。

## 用 AI Horde 快速开始
> 💡 **肆说**：完全免费！不用注册任何东西，填个0000000000就能连。但可能要排队等一会儿。

1. 按照 [安装指南](../installation/index.md) 安装并启动 SillyTavern
2. 在 SillyTavern 的引导界面，输入你的人设名字。这个名字会在聊天中使用
3. 点击顶部栏的 API 连接按钮
4. 输入 AI Horde 的 API 密钥。你可以暂时用 `0000000000`，或者从 [AI Horde](https://aihorde.net/) 获取免费密钥
5. 选择几个 AI 模型。随便挑几个排在前面的就行，之后随时可以换
6. 关闭 API 连接窗口。在底部聊天框输入消息，按回车
7. AI 会在几秒内回复。你可以继续 [聊天](chatting.md)。成功！

## 用 OpenAI 快速开始
> 💡 **肆说**：国内需要梯子+海外支付。效果最好但要花钱。

### 安装 SillyTavern

按照 [安装指南](../installation/index.md) 安装并启动 SillyTavern。

### 获取 OpenAI 访问权限

1. 注册 OpenAI 账号
2. 访问 [https://platform.openai.com](https://platform.openai.com/)
3. 点击右上角账号图标，然后 View API Keys
4. 点击 "Create new secret key"。**立刻复制保存好！绝对不要分享这个密钥！拥有它的人可以用你的账号消费 GPT！**

### 配置 SillyTavern 使用你的 API

1. 在 SillyTavern 顶部栏，点击 API 连接
2. API 选择 Chat Completion (OpenAI)
3. Chat Completion Source 选择 OpenAI
4. 粘贴上一步保存的 API 密钥
5. 点击连接按钮，确认显示 Valid
6. 默认使用 GPT-4 Turbo，你可以选其他模型，但先了解价格

### 测试你的设置

1. 在顶部栏点击最右边的角色管理
2. 选择一个已有角色，比如 Seraphina
3. 在底部文本框写点什么给 Seraphina，按回车或点发送

如果一切正确，几秒后 Seraphina 就会回复你。
