# 🃏 角色卡管理
> 💡 **肆说**：角色卡就是你跟AI聊天的"入口"，每个角色卡定义了一个AI人格。

角色是你创建和管理的 AI 身份。每个角色有名字、个性和对话历史。你可以创建无限角色，随时切换。

角色可用于单人聊天，或添加多个角色到群聊让它们互动。

## 角色管理面板

从导航栏打开角色面板。

### 面板控制
- **锁定面板**：交互时保持面板打开
- **角色列表**：返回列表视图
- **HotSwap 栏**：快速访问收藏角色

### 角色列表功能
- 创建新角色、从文件导入、从URL导入、创建群聊
- 搜索和排序：按字母、时间、使用频率、大小等
- 按类型或标签过滤
- 收藏、群聊、标签文件夹

### 角色创建/编辑面板
- 头像上传和预览
- Token 计数
- 聊天统计
- 标签管理
- 快捷操作：收藏、高级定义、角色知识库、聊天知识库、导出、复制、删除

### 高级定义面板
> 💡 **肆说**：写卡进阶功能都在这——提示词覆盖让角色有独立指令，对话示例教AI怎么说话。
- **提示词覆盖**：主指令覆盖、后历史指令覆盖
- **创建者元数据**：创建者名称、版本、备注
- **角色个性**：个性摘要、场景、角色备注(@ Depth)、健谈度
- **对话示例**
