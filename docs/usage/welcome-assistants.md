# 欢迎页助手
> 💡 **肆说**：小功能，可以设置一个角色当欢迎页的"看板娘"，纯装饰用。

SillyTavern有一个欢迎屏幕功能，可以用指定的"助手"角色来迎接你。当你在没有活跃聊天的情况下启动SillyTavern，或者关闭最后一个聊天会话后，这个屏幕就会出现。

##### 注意

如果你在应用启动时没有看到欢迎屏幕，请确保在**用户设置**面板的"聊天/消息处理"部分中禁用了"自动加载上次聊天"选项。

## 欢迎屏幕

当没有活跃聊天时，欢迎屏幕提供以下元素：

- **SillyTavern版本号**：显示应用的Logo和当前版本。

- **快速链接**：轻松访问文档、GitHub仓库和Discord服务器。

- **临时聊天按钮**：允许你快速与默认中立助手开始一个新的临时聊天会话。

- **最近聊天区域**：列出你最近的对话以便快速访问。

## 临时聊天

临时聊天按钮允许你快速开始一个新的聊天会话，而不会将其保存到聊天历史中。一旦你关闭或切换到另一个聊天，这个聊天就会被删除。

- **保存**按钮将允许你将临时聊天导出为JSONL文件。

- **加载**按钮将允许你恢复之前保存的临时聊天文件。

## 什么是欢迎页助手？

欢迎页助手是你选择在欢迎屏幕上展示的角色。

### 设置和取消助手

1. 导航到**角色管理**面板。

2. 找到你想设置为助手的角色，点击"更多..."并选择**"设置/取消为欢迎页助手"**。

3. 角色名称旁边会出现图标表示已设置。

取消时重复相同操作即可。SillyTavern将恢复使用默认助手。

## 默认助手

SillyTavern会自动创建一个名为"Assistant"的默认角色。如果你没有设置特定角色作为欢迎页助手，将使用此默认助手。你可以自由自定义它。
