# 斜杠命令（Slash Commands）
> 💡 **肆说**：斜杠命令是SillyTavern的"控制台"，输入/开头的命令可以快速执行各种操作。不用全记，用的时候查就行。

**这不是完整列表，因为很少更新。**

要获取你的实例中可用的最新命令列表，请在任意SillyTavern聊天中使用 `/help slash` 命令。

- `/help` – 显示帮助消息
- `/api` - 切换到不同的API
- `/sys` - 以系统旁白身份发送中立消息
- `/send` - 以用户身份发送消息但不触发生成
- `/sendas` – 以特定角色身份发送消息
- `/comment` - 向聊天添加AI不可见的评论
- `/impersonate` - 让AI以你的身份写作，可带可选提示词
- `/persona` - 设置你的用户人设
- `/imagine` - 从文本提示词生成图片
- `/bg` - 按文件名设置背景图片

📖 原文链接：[Slash commands](https://docs.sillytavern.app/usage/core-concepts/slashcommands/)
