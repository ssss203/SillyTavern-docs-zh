# 标签（Tags）
> 💡 **肆说**：标签就是给角色卡"贴标签"分类的，角色多了之后很有用。角色少的时候可以先不管。

角色卡和群组可以分配零个或多个标签。标签对于按照主题、质量、来源等方式快速整理不断增长的集合很有用。

## 打标签

- 导入时嵌入标签
- 从角色管理面板打开角色卡来分配标签
- 批量打标签：点击"批量编辑角色"按钮，选择角色卡，右键点击，选择"标签"

##### 注意：群组不能批量打标签。

## 管理

打开角色管理面板，点击"管理标签"按钮（齿轮图标）。可备份和恢复所有标签信息。

## 导入角色卡时导入标签

从外部导入角色卡时会提示是否导入其中包含的标签。标签不是功能必需的，仅用于组织。

## 过滤角色卡

创建标签后，点击标签切换过滤状态：显示带有此标签 → 仅显示不带有此标签 → 重置过滤。

## 标签作为文件夹

需先在用户设置中启用。每个标签可设为"打开文件夹"、"关闭文件夹"或重置状态。
