# 👤 人设（Personas）
> 💡 **肆说**：人设就是"你是谁"。想让AI叫你不同名字、扮演不同身份时，切换人设就行，不用每次改设置。

## 什么是人设？

人设是你在聊天中使用的身份——基本上是你的显示名、头像和可选描述文字的组合。人设让你轻松切换你说话的"角色"，不用每次手动改用户名/头像。

## 如何创建人设？

1. 打开人设管理面板
2. 用创建按钮创建空白人设并命名
3. 在人设列表选择新创建的人设
4. 在右侧填写描述并通过"更改人设图片"设置头像（都是可选的）
5. 人设就可以在聊天中使用了

### 将角色转换为人设

打开角色，选"More..."点击"Convert to Persona"。会创建同名同描述的人设。角色不会被删除。

## 人设描述
> 💡 **肆说**：人设描述可以写你自己的设定，比如"我是个沉默寡言的剑客"，AI会根据这个跟你互动。

每个人设可存储自定义文字描述。注入位置取决于人设管理面板的位置设置：
- None（禁用）
- In Story String / Prompt Manager（默认）
- Top/Bottom of Author's Note
- In Chat @ Depth

## 人设连接/锁定

1. **聊天锁定** - 人设锁定到当前聊天
2. **角色锁定** - 人设锁定到特定角色
3. **默认人设** - 没有其他锁定时使用的人设

## 斜杠命令

- `/persona-lock type=<type?>` - 锁定人设（chat/character/none）
- `/persona <name>` - 快速切换人设
- `/persona-sync` - 将聊天中所有用户消息重新归属到当前人设
