# 用户设置（User Settings）
> 💡 **肆说**：这里控制SillyTavern的各种行为偏好，大部分用默认值就行，不用全改。

## 通用设置

### UI 语言

选择 SillyTavern 用户界面的显示语言。

### 软件版本

显示当前安装的 SillyTavern 版本号。

### 账号管理

#### 适用于：[多用户模式](https://docs.sillytavern.app/administration/multi-user/)

多账号功能需要在 config.yaml 中将 `enableUserAccounts` 设为 true。

- **修改密码** — 更改当前账号密码
- **设置快照** — 访问和查看 `settings.json` 文件的备份，可以创建或恢复快照
- **下载备份** — 下载用户数据文件夹的压缩包
- **重置设置** — 恢复出厂默认设置，同时保留其他数据（角色、聊天）
- **删除所有内容** — 完全清除账号和恢复出厂设置

#### 管理面板

##### 适用于：[多用户模式](https://docs.sillytavern.app/administration/multi-user/)

多账号功能需要在 config.yaml 中将 `enableUserAccounts` 设为 true。

选择 **管理用户** 来查看和管理现有用户账号。

##### 用户资料

- 自定义头像管理（上传/移除）
- 显示名称和句柄
- 角色和状态信息
- 账号创建日期
- 密码保护状态

##### 账号控制

- 编辑显示名称
- 启用账号
- 禁用账号
- 提升为管理员
- 降级为普通用户

##### 管理操作

- 下载用户数据备份
- 更改用户密码
- 删除账号

##### 新用户

选择 **新用户** 来创建新的用户账号。

- 显示名称*（例如 "John Snow"）
- 用户句柄*（仅限小写字母、数字和连字符）
- 密码（可选）
- 密码确认

创建新用户会自动在 /data/ 目录下使用用户的句柄作为文件夹名生成一个子文件夹。

#### 注销

##### 适用于：[多用户模式](https://docs.sillytavern.app/administration/multi-user/)

退出当前会话。

### 设置搜索
> 💡 **肆说**：设置太多找不到？用搜索栏！输入关键词就能定位。

一个方便的搜索栏，帮助你快速找到特定设置：

- 输入任意关键词来过滤和高亮用户设置中的设置
- 搜索设置名称和描述
- 帮助更高效地导航复杂设置

有关**用户设置**中此部分设置的更多信息，请参见 [UI 自定义](https://docs.sillytavern.app/usage/core-concepts/uicustomization/#ui-theme)。

## 角色处理

- **角色列表副标题**：选择在[角色](https://docs.sillytavern.app/usage/core-concepts/characterdesign/)列表中角色名称下方显示的额外信息：
  - 角色版本
  - 创建者
- **导入卡片标签**：控制导入角色卡片时标签的处理方式：
  - 询问 — 每次导入显示对话框
  - 无 — 不导入任何标签
  - 全部 — 导入所有标签
  - 已存在 — 仅导入已存在的标签
- **高级角色搜索**：启用后，使用模糊匹配并搜索所有角色数据字段，而不仅仅是名称。
- **优先角色提示词**：如果启用，当可用时使用角色卡的系统提示词覆盖。
> 💡 **肆说**：推荐开启！开了之后角色卡自己的系统提示词会覆盖全局的，让每个角色有独立指令。
- **优先角色指令**：如果启用，当可用时使用角色卡的历史后指令覆盖。
> 💡 **肆说**：推荐开启！同上，让角色的后历史指令优先于全局设置。
- **永不调整头像大小**：防止裁剪/调整导入角色图片的大小。禁用时，图片调整为 512x768。
- **显示头像文件名**：在角色列表中显示角色头像的实际文件名。
- **无剧透模式**：在编辑面板中用剧透按钮隐藏角色定义。

## 杂项

- **重新加载聊天**：重新加载并重绘当前聊天。
- [**调试菜单**](https://docs.sillytavern.app/usage/user-settings/#debug-menu)：访问调试选项。
- **平滑流式输出（Smooth Streaming）**：通过逐字母显示文本来平滑流式生成。包括速度控制滑块。要从平滑流式输出中排除推理块，启用"排除'思考中...'"选项。
> 💡 **肆说**：开了之后AI回复会像打字一样逐字出现，体验更流畅。推荐开启。
- **流式渐入（Stream Fade-In）**：对流式文本应用渐入效果。可与平滑流式输出同时使用或单独使用。
- [**消息声音**](https://docs.sillytavern.app/usage/core-concepts/uicustomization/#message-sound)：消息生成完成时播放声音。
  - **仅后台声音**：仅在浏览器标签页未聚焦时播放声音。
- **宽松 API URL**：降低 API URL 的格式要求。
- **知识书导入对话框**：导入带有嵌入知识书的角色时显示导入对话框。
- **自动选择输入文本**：点击时自动选择某些输入字段中的文本。
- **Markdown 快捷键**：启用 Markdown 格式化的键盘快捷键。
- **恢复用户输入**：页面刷新时保留未保存的用户输入。
- **MovingUI**：允许通过拖动重新定位 UI 元素（仅限 PC）。
  - **重置** 按钮恢复默认位置
  - 用于保存/加载 UI 布局的预设系统

## 聊天/消息处理

### 消息显示设置

控制消息在聊天界面中的加载和显示方式。这些设置影响整体聊天体验和性能。

- **加载消息数**：分页前加载的聊天历史消息数（0 = 全部）
- **流式 FPS**：流式文本的更新速度（5-100 FPS）
- **示例消息行为**：
  - 渐进推出
  - 始终包含示例
  - 从不包含示例
- **图片滑动行为**（控制画廊样式中图片的滑动操作）：
  - 生成新图：允许使用[图片生成](https://docs.sillytavern.app/extensions/stable-diffusion/)扩展生成新图片
  - 循环：在现有图片间循环，在两端环绕

### 输入与响应控制

决定消息如何发送以及 AI 如何继续其响应的设置。

- **回车发送**：在禁用、自动（PC）或启用之间选择
- **"发送"继续**：使用发送按钮继续 AI 响应
- **快速"继续"按钮**：显示延长 AI 最后一条消息的按钮
- **快速"模仿"按钮**：显示单消息角色模仿的按钮
- **滑动（Swipes）**：显示替代 AI 响应的箭头按钮（PC 和移动端）
- **手势**：启用手势滑动生成（仅限移动端）

### 自动管理

帮助管理聊天流程和内容的自动化功能。

- **自动加载上次聊天**：启动时自动加载最近的聊天
- **自动滚动聊天**：自动滚动到最新消息
- **自动保存消息编辑**：无需确认即可保存消息编辑
- **确认消息删除**：删除消息前提示确认
- **自动修复 Markdown**：自动纠正 Markdown 格式

#### 自动滑动（Auto-swipe）
> 💡 **肆说**：让AI自动重试不满意的回复——太短的、包含特定词的都可以自动跳过重新生成。

根据可配置的标准自动拒绝并重新生成 AI 消息。

- **启用自动滑动**：自动滑动功能的主开关
- **最小生成消息长度**：如果消息短于此值则触发自动滑动
- **黑名单词**：可触发自动滑动的词列表，用逗号分隔
- **黑名单词计数滑动**：必须检测到的最少黑名单词数才触发自动滑动

#### 自动继续（Auto-Continue）
> 💡 **肆说**：AI写一半停了？开这个让它自动继续写。但注意不会让AI写更多，只是把截断的补完。

如果模型在达到一定长度之前停止，自动继续响应。

这使你的 AI 可以分多部分编写长响应，这样你可以设置较短的[响应长度设置](https://docs.sillytavern.app/usage/common-settings/#response-tokens)同时仍能获得长回复。

它不会让 AI 写出比原本更多的内容。要求 AI 继续一条它认为"已完成"的消息通常不起作用。参见[如何让 AI 写更多？](https://docs.sillytavern.app/usage/faq/#how-to-make-the-ai-write-more)了解其他想法。

- **启用自动继续**：自动继续的主开关
- **允许聊天补全 API**：为聊天补全 API 端点启用自动继续功能
- **目标长度（令牌）**：期望的消息长度（令牌数）— 如果消息短于此值将触发继续（0-1024）

### 消息格式与显示

控制消息的格式化方式和显示的内容。

- **禁止外部媒体**：阻止外部域的嵌入媒体
- **显示 {{char}}: 在响应中**：如果生成时保留了角色名前缀则在响应中保留
- **显示 {{user}}: 在响应中**：如果生成时保留了用户名前缀则在响应中保留
- **实验性宏引擎**：启用支持嵌套的高级宏处理
- **显示响应中的标签**：允许（某些）HTML 标签在响应中作为 HTML 显示
- **宽松群组消息修剪**：允许 AI 在群组聊天中替其他角色说话，而不是停止响应生成
- **显示群组聊天队列**：在群组聊天的角色列表中显示响应顺序
- **固定问候消息样式**：始终渲染问候语中的样式标签，即使消息因懒加载而卸载

### 提示词检查与调试

- **将提示词记录到控制台**：将提示词输出到浏览器控制台
- **请求令牌概率**：从 API 请求 AI 响应的令牌概率。在可用的情况下，可以在[令牌概率](https://docs.sillytavern.app/usage/chatting/#token-probabilities-panel)中查看。

### 自动补全（AutoComplete）

- 自动隐藏详情
- 匹配样式（开头/包含/模糊）
- 视觉样式（主题/深色/浅色）
- 键盘选择选项
- 字体缩放
- 宽度控制

## STscript 设置

[STscript 解析器](https://docs.sillytavern.app/usage/st-script/#parser-flags)的配置选项。

### STRICT_ESCAPING

- 引号值中的管道不需要转义。
- 符号前的反斜杠可以转义以提供字面反斜杠后跟功能符号。

详见[严格转义](https://docs.sillytavern.app/usage/st-script/#strict-escaping)。

### REPLACE_GETVAR

有助于避免当变量值包含可能被解释为宏的文本时发生双重替换。

详见[替换变量宏](https://docs.sillytavern.app/usage/st-script/#replace-variable-macros)。

## 清理菜单
> 💡 **肆说**：用久了SillyTavern会攒一堆没用的文件，定期清理能省不少空间。但删了就不能恢复了，先备份！

清理菜单提供了一个数据维护工具，帮助你识别和移除 SillyTavern 安装中不必要的文件。此功能帮助保持数据目录的整洁并可以释放大量磁盘空间。

##### 重要警告

清理工具将永久删除文件。**此操作无法撤销！**

手动上传到 `/data/user/files/` 和 `/data/user/images/` 目录的文件，如果未与聊天消息或数据银行条目关联，将被删除。

如果不确定，使用清理菜单前请备份数据。

### 如何使用清理功能

1. 点击**杂项**部分下的**清理**按钮
2. 点击**扫描**来分析你的安装。这可能需要一些时间，取决于数据目录的大小
3. 查看找到的文件类别
4. 使用**查看**在删除前预览文件内容
5. 使用**下载**在删除前保存文件
6. 根据需要删除单个文件或整个类别

### 清理类别

清理工具扫描的松散文件分为以下类别：

#### 文件

- **查找内容**：未与聊天消息或数据银行条目关联的文件
- **位置**：`/data/<user-handle>/user/files/`
- **风险**：⚠️ **将删除未引用的手动上传**
- **何时清理**：如果不需要未引用的文件，可以安全删除

#### 图片

- **查找内容**：未与聊天消息关联的图片
- **位置**：`/data/<user-handle>/user/images/`
- **风险**：⚠️ **将删除未引用的手动上传**
- **何时清理**：如果不需要未引用的图片，可以安全删除

#### 聊天

- **查找内容**：与已删除角色关联的聊天文件
- **位置**：`data/<user-handle>/chats/`
- **风险**：⚠️ **孤立聊天将永久丢失**
- **何时清理**：如果你有意删除角色且不再需要其聊天历史，可以安全删除

#### 群组聊天

- **查找内容**：与已删除群组关联的聊天文件
- **位置**：`data/<user-handle>/group chats/`
- **风险**：⚠️ **孤立群组聊天将永久丢失**
- **何时清理**：如果你有意删除群组且不再需要其聊天历史，可以安全删除

#### 头像缩略图

- **查找内容**：已缺失或已删除角色的头像缩略图
- **位置**：`data/<user-handle>/thumbnails/avatar`
- **风险**：✅ **可以安全删除** — 需要时会自动重新生成
- **何时清理**：始终可以安全清理，有助于释放空间

#### 背景缩略图

- **查找内容**：已缺失或已删除背景的缩略图
- **位置**：`data/<user-handle>/thumbnails/bg`
- **风险**：✅ **可以安全删除** — 需要时会自动重新生成
- **何时清理**：始终可以安全清理，有助于释放空间

#### 聊天备份

- **查找内容**：自动生成的聊天备份
- **位置**：`data/<user-handle>/backups/chat_*`
- **风险**：⚠️ **备份文件将永久丢失**
- **何时清理**：考虑保留最近的备份，较旧的可以安全删除

#### 设置备份

- **查找内容**：自动生成的设置备份
- **位置**：`data/<user-handle>/backups/settings_*`
- **风险**：⚠️ **设置备份文件将永久丢失**
- **何时清理**：考虑保留最近的备份，较旧的可以安全删除

## 调试菜单
> 💡 **肆说**：你不是开发者就别碰这些，大部分功能你用不到。

##### 这些功能仅面向高级用户。

除非你完全理解其后果，否则不要使用。

调试菜单提供了用于故障排除、维护和开发的功能。这些功能应谨慎使用，因为它们可能显著影响你的 SillyTavern 安装。

由于扩展可以添加调试功能，可用选项会根据你安装的扩展而有所不同。

### 翻译与区域设置功能

- **获取缺失翻译**：分析当前区域设置（如果选择英语则分析所有区域设置）的缺失翻译，并将结果输出到浏览器控制台
- **应用区域设置**：通过重新应用选定的语言设置来强制刷新

### 缓存与存储管理

- **清除 WebSearch 缓存**：从本地缓存中移除所有存储的搜索结果
- **清除所有向量索引**：完全移除所有来源中存储的向量
- **重置令牌缓存**：清除存储的令牌计数，强制重新令牌化所有聊天
- **删除逐项提示词**：从本地存储中移除所有逐项提示词

### 数据与统计

- **刷新统计文件**：使用现有聊天数据重建统计文件
- **回填令牌计数器**：重新计算当前聊天中所有消息的令牌数
  - 在切换不同令牌器的模型时有用
  - 完成后触发聊天重新加载
  - 仅视觉更改，不修改聊天内容

### API 与扩展测试

- **更改 Mancer 基本 URL**：修改 Mancer API 服务器的基本 URL
- **测试 WebSearch 扩展**：使用当前设置执行测试搜索
- **发送生成请求**：使用当前选择的 API 测试文本生成

### 系统与调试工具

- **强制新手引导**：重新启动新手引导过程
- **切换事件追踪**：启用/禁用用于调试的事件跟踪
- **复制 ST 设置**：[开发中] 将系统配置数据复制到剪贴板用于错误报告

每个功能可以使用其描述下方的"执行"按钮来执行。在使用这些工具之前考虑备份数据，因为某些操作无法撤销。
