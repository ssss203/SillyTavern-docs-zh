# 视觉小说模式（Visual Novel Mode）
> 💡 **肆说**：想让聊天界面像视觉小说游戏一样？开这个模式！角色立绘会显示在聊天背景上，氛围感拉满。

视觉小说模式是 SillyTavern 中的一种特殊屏幕布局，允许你与带有精灵图（Sprites，或角色卡图片）的角色聊天，类似于《心跳文学部》、《灰色果实》、《Fate/stay night》等著名视觉小说游戏的风格。

## 切换视觉小说模式

### 启用视觉小说模式

视觉小说模式内置于 SillyTavern 中，可以通过进入*用户设置*（用户设置图标），在*无文字阴影*下方勾选**视觉小说模式**来切换。

### 禁用视觉小说模式

禁用视觉小说模式的步骤与启用相同。取消勾选视觉小说模式即可回到正常的聊天界面。

##### 关于 VN 模式与 VN 扩展

某些扩展（如 Prome VN 扩展）在使用它们各自的 VN 模式时会自动切换"视觉小说模式"。从*用户设置*菜单启用/禁用 VN 模式也会影响这些扩展。

## 视觉小说 UI

在视觉小说模式中，UI 会略有改变以适应角色精灵图（或角色卡图片），它显示在中央。但在有多个角色的群组聊天中，角色精灵图会自动分散排列，为彼此腾出空间。

### VN 模式与 MovingUI

要切换 MovingUI，进入*用户设置*并勾选**MovingUI**。请注意，此功能**仅**在桌面端可用。

如果在*用户设置*中启用了 **MovingUI**，精灵图（或角色卡图片）可以移动，如果你想移动它们或放置在屏幕上更特定的区域。

##### 关于精灵图大小

如果你的角色精灵图比较大，使用 MovingUI 移动某些精灵图会比较困难，因为拖动精灵图的按钮可能被现有精灵图覆盖。你可能需要比正常更多次地移动它们，特别是屏幕上有更多角色时才能获得更好的放置效果。

## 获取角色精灵图
> 💡 **肆说**：没有精灵图也能用——直接用角色头像就行，虽然不像立绘但至少有图看。

获取角色精灵图可以通过在互联网上搜索现有精灵图来完成，例如来自视觉小说或使用视觉小说功能游戏（如 DDLC 或 CounterSide）的现有角色。如果你想要精灵图的角色没有现成的精灵图，你还有以下几个选择。

1. 在角色帖子中搜索精灵图 ZIP 包或精灵图包链接。

某些机器人创建者可能会随机器人发布精灵图包（在同一帖子中或精灵图频道中）。搜索这些帖子，看看是否有人已经制作了你想要的角色精灵图。

2. 使用 LoRA 和 Stable Diffusion 创建自己的精灵图。

从零开始生成精灵图非常耗时（特别是如果你的角色没有现成的 LoRA 和/或你想使用的 Stable Diffusion 模型没有），并且需要不错的硬件才能生成，尤其是如果你计划制作 28 种精灵表情而不是 6 种，以及如果你使用 SDXL 和/或将精灵图放大到更高分辨率。

3. 使用角色卡图片。它可能不像精灵图，但至少你在屏幕上有东西可以看。但是，多个角色卡不能在 VN 模式中使用。

##### 使用 Prome 视觉小说扩展的角色卡图片

在 Prome 视觉小说扩展 1.0.6+ 中，有一个名为"将角色卡模拟为精灵图"的功能，允许你在群组聊天中同时使用精灵图和非精灵图角色，方法是将其角色卡作为精灵图使用。

## VN 扩展

### Prome 视觉小说扩展

Prome 视觉小说扩展是来自 Bronya Rand 和 Prometheus 的认可第三方扩展，进一步增强 SillyTavern 中的视觉小说体验，功能包括信箱模式（Letterbox Mode，使视觉小说 UI 更有"电影感"）、带角色精灵图变暗的聚焦模式（Focus Mode）、传统 VN 模式（聊天中只显示最后一条消息）以及更多计划中的功能！

要安装 Prome 视觉小说扩展，你可以通过进入"下载扩展与资产"并找到 *Prome Visual Novel Extension* 来安装，或按照 [Prome 视觉小说扩展](https://github.com/Bronya-Rand/Prome-VN-Extension?tab=readme-ov-file#installation-and-usage) Github 页面上的安装说明进行操作。调整 Prome 的设置可以在*扩展* → **Prome (Visual Novel Extension)** 中找到，或通过 🪄（魔法棒）菜单。
