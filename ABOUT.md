# SillyTavern 官方文档中文小白板

> 给看不懂官方英文文档、也啃不动直译版的小白用的说明书。

## 这是啥

SillyTavern 官方文档（docs.sillytavern.app）的中文小白板版本。覆盖全部 89 页官方文档，信息量不删减，但用大白话写。

每一页底部都附有 📖 原文链接，想看专业原文点过去就行。

## 跟直译站的区别

| | 直译站 | 小白板 |
|---|---|---|
| 信息量 | 完整 | 完整，不删减 |
| 语言风格 | 英文直译，专业术语保留 | 大白话，术语加人话解释 |
| 额外注释 | 无 | 💡肆说：使用场景提示、国内可用性、参数推荐值、小白避坑 |
| 适合谁 | 有技术背景的开发者 | AI外行、写角色卡的新手 |

## 💡肆说 是什么

老师注释。告诉你：
- 这个参数写卡会不会用到
- 这个API国内能不能直连
- 不知道选啥时的推荐默认值
- 哪些可以先跳过、哪些是核心必读

## 目录结构

完全对齐官方文档：

```
docs/
├── installation/       # 安装指南
├── usage/              # 使用指南
│   ├── api-connections/ # API连接（各平台配置）
│   ├── core-concepts/   # 核心概念（写卡必看）
│   ├── prompts/         # 提示词与预设
│   └── user-settings/   # 用户设置
├── extensions/          # 扩展功能（生图/语音/翻译等）
├── administration/      # 管理员指南（简版）
└── for-contributors/    # 贡献者文档（简版）
```
